# yuuki-public-media

Public image host for Notion embeds. Yuuki pushes screenshots here so they can be rendered inline on Notion pages via raw.githubusercontent.com URLs.

Files in here are public — only push content that's already publicly visible (e.g. screenshots of public Instagram reels, public blog excerpts).

Populated by `yuuki-runtime/notion-image-bridge.sh`. Do not edit by hand.
